      MediaTekDocuments

      MediaTekDocuments est une application de gestion de documents destinée aux médiathèques. Elle permet de gérer les livres, DVD, revues, abonnements, et commandes via une interface WinForms connectée à une API REST.

      Ce projet a été réalisé dans le cadre du BTS SIO SLAM :

      • model : classes métiers
      • manager : logique métier
      • controller : coordination entre les vues et les données
      • dal : accès aux données via l’API REST
      • view : interface graphique utilisateur (WinForms)

      Namespaces

      MediaTekDocumentsGère l’ensemble et contient le point d'entrée de l'application MediaTekDocuments
      MediaTekDocuments.controllerContient les classes contrôleurs utilisées pour faire le lien entre l'interface graphique et les classes métiers
      MediaTekDocuments.dalGère l'accès aux données via l'API REST. Contient la logique de communication avec le serveur distant et le traitement des réponses JSON.
      MediaTekDocuments.managerContient les traitements spécifiques non liés à l'interface graphique ou à l'accès aux données et la communication avec L'API REST.
      MediaTekDocuments.modelContient les classes métier représentant les objets dans l'application : documents, commandes, abonnements, utilisateurs, etc.
      MediaTekDocuments.viewContient les classes liées à l'interface graphique (WinForms) permettant l'affichage, la saisie et la gestion des interactions utilisateur.
